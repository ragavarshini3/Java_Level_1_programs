# Java Level 1 Problem Solving

This repository contains solutions for Java Level 1 programming problems.

The problems focus on:

- Basic Input and Output
- Arithmetic Operations
- Quotient and Remainder
- Digit Extraction
- Sum of Digits
- Number Reversal
- Digit Manipulation
- Logical Thinking
- Problem Solving Skills

---

## Topics Covered

### Basic Arithmetic
- Add a number
- Subtract a number
- Multiply a number
- Division
- Modulus

### Digit Operations
- Extract one's digit
- Extract ten's digit
- Extract hundred's digit
- Sum of digits
- Reverse digits

### Number Manipulation
- Replace digits
- Swap digits
- Modify specific positions
- Generate new numbers from existing digits

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/java-level-1.git
```

2. Navigate to the project folder

```bash
cd java-level-1
```

3. Compile the program

```bash
javac Problem01.java
```

4. Run the program

```bash
java Problem01
```
